
### IP 와 서브넷마스크


> IP

```

인터넷상에서 데이터를 주고받기 위해 필요한 컴퓨터 고유의 주소를 IP라고 함.
IP에는 IPv4 와 IPv6 표기방식이 있음.

(십진수)       -> (이진수)
192.168.10.10 -> 1100 0000. 1010 1000. 0000 1010. 0000 1010

32비트 총 4바이트로 구성되어 있으며, 1바이트를 옥텟이라고 부름.
IP는 총 5개의 클래스 (A, B, C, D, E 클래스)로 구분되며, D/E 클래스는 실제 IP로 활용하지 않음.


- A 클래스
0xxx xxxx. xxxx xxxx. xxxx xxxx. xxxx xxxx.
첫번째 옥텟의 비트가 0으로 고정되므로 아래의 IP 주소만 가질 수 있음.

0000 0000. 0000 0000. 0000 0000. 0000 0000 ~ 0111 1110. 1111 1111. 1111 1111. 1111 1111.
0.0.0.0 ~ 127.255.255.255

이 클래스는 대큐모 네트워크에 적합함.
네트워크 주소 8비트 / 호스트 주소 24비트


- B 클래스
10xx xxxx. xxxx xxxx. xxxx xxxx. xxxx xxxx.
첫번째 옥텟의 비트가 10으로 고정되므로 아래의 IP 주소만 가질 수 있음.

1000 0000. 0000 0000. 0000 0000. 0000 0000. ~ 1011 1111. 1111 1111. 1111 1111. 1111 1111.
128.0.0.0 ~ 191.255.255.255

네트워크 주소 16비트 / 호스트 주소 16비트


- C 클래스
110x xxxx. xxxx xxxx. xxxx xxxx. xxxx xxxx.
첫번째 옥텟의 비트가 110으로 고정되므로 아래의 IP 주소만 가질 수 있음.

1100 0000. 0000 0000. 0000 0000. 0000 0000. ~ 1101 1111. 1111 1111. 1111 1111. 1111 1111.
192.0.0.0 ~ 223.255.255.255

네트워크 주소 24비트 / 호스트 주소 8비트


- D 클래스
1110 xxxx. xxxx xxxx. xxxx xxxx. xxxx xxxx.
첫번째 옥텟의 비트가 1110으로 고정되므로 아래의 IP 주소만 가질 수 있음.

1110 0000. 0000 0000. 0000 0000. 0000 0000. ~ 1110 1111. 1111 1111. 1111 1111. 1111 1111.
224.0.0.0 ~ 239.255.255.255

멀티캐스트용 대역으로 일반적인 IP주소로 할당하지 않음.


- E 클래스
1111 xxxx. xxxx xxxx. xxxx xxxx. xxxx xxxx.
첫번째 옥텟의 비트가 1111로 고정되므로 아래의 IP 주소만 가질 수 있음.

1111 0000. 0000 0000. 0000 0000. 0000 0000. ~ 1111 1111. 1111 1111. 1111 1111. 1111 1111.
240.0.0.0 ~ 255.255.255.255


- 사설IP 대역
A클래스 사설 IP 주소 : 10.0.0.0 ~ 10.255.255.255
B클래스 사설 IP 주소 : 172.16.0.0 ~ 172.31.255.255
C클래스 사설 IP 주소 : 192.168.0.0 ~ 192.168.255.255

0.0.0.0, 255.255.255.255와 같이 네트워크 시작주소, 브로드캐스트용 주소는 IP주소로 할당하지 않음.
* 브로드캐스트 주소 : xxx.xxx.xxx.255

```

> 사설IP VS 공인IP

```
- 공인IP (public IP)
ISP(인터넷 서비스 공급자)가 제공하는 IP주소로 외부에 공개되어 있는 IP주소임.
  * 공인IP는 전세계에서 유일함.
  * 공인IP주소는 외부에 공개되어있으므로 인터넷에 연결된 다른 PC로부터 접근이 가능함.
    따라서 공인IP주소를 사용하는 경우 방화벽 등의 보안프로그램을 설치할 필요가 있음.

    * 할당 주체 : ISP 인터넷 서비스 공급자
    * 할당 대상 : 개인 또는 회사의 서버(라우터)
    * 고유성 : 인터넷 상에서 유일한 주소
    * 공개 여부 : 내/외부 접근 가능



- 사설IP (Pri)
일반 가정이나 회사 내 등에 할당된 네트워크의 IP주소이며, 로컬IP / 가상IP라고 함.
IPv4의 주소 부족으로 인해 서브넷팅된 IP이기 때문에 라우터에 의해 로컬 네트워크 상의 PC나 장치에 할당함.

    * 할당 주체 : 라우터(공유기)
    * 할당 대상 : 개인 또는 회사의 기기
    * 고유성 : 하나의 네트워크 안에서 유일
    * 공개 여부 : 외부 접근 불가능

** 사설 IP주소만으로는 인터넷에 직접 연결할 수 없음. 라우터를 통해 1개의 공인IP만 할당하고,
라우터에 연결된 개인 PC는 사설 IP를 각각 할당 받아 인터넷에 접속할 수 있게됨.

```

> 서브넷마스크 / 게이트웨이

```
- 서브넷
IP주소에서 네트워크 영역을 부분적으로 나눈 부분망, 부분네트워크를 뜻함.

- 서브넷마스크
서브넷을 만들 때 사용되는 것으로 로컬 네트워크 내부에서 접속한 호스트의 IP대역을 외부 네트워크와 명확하게 구분할 수 있는 수단을 표준화한 것임.
서브넷을 구분하는 방법 중 하나임.

- 서브넷팅
클래스 별 기본 서브넷 마스크를 쓰지 않고 서브넷 마스크를 커스텀으로 변경하여 네트워크 부분을 늘리고 호스트 부분을 줄이는 조정을 하는 것


- 게이트웨이
컴퓨터 네트워크에서 서로 다른 통신망, 프로토콜을 사용하는 네트워크 간의 통신을 가능하게 하는 컴퓨터나 소프트웨어를 두루 일컫는 용어, 즉 다른 네트워크로 들어가는 입구 역할을 하는 네트워크 포인트임.
```
