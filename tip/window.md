> cmd 명령어

```
calc = 계산기
cmd = 도스명령프롬프트
control = 제어판
(control mouse = 제어판의 마우스 속성, control keyboard = 키보드 속성, citrus님 팁)
notepad = 메모장
mspaint = 그림판
regedit = 레지스트리 편집기
winword = ms 워드
powerpnt = ms 파워포인트
excel = ms 엑셀
onenote = ms 원노트
snippingtool = 캡쳐 도구
iexplore = 인터넷 익스플로러
downloads = 다운로드 폴더
ncpa.cpl = 네트워크 연결 (캐슬워터님 팁)
main.cpl = 마우스 속성 (CROWN님 팁)
appwiz.cpl = 프로그램 제거 또는 변경 (shinysun님 팁)
services.msc = 실행중 서비스확인


-- 실행창에서 활용할 수 있는 명령어(경로) 만들기
또 실행창을 활용할 수 있는 두번째 방법은  C:\windows\system32 폴더를 활용하여 명령어를 직접 만드는 방법입니다.

C:\windows\system32 폴더에 파일을 넣어두면 길게 프로그램의 위치를 입력하지 않고 파일명만 입력하여도 프로그램 실행이 가능합니다. 자주 사용하는 실행파일을 바로가기로 만들어 system32 폴더에 위치시키면, 실행창을 연 후, 프로그램 명만 넣어도 실행이 가능해서 굉장히 유용합니다.

(exe 파일의 경우에는 확장자명을 적지 않아도 실행이 가능하나, 다른 확장자의 경우에는 확장자까지 적어주어야 실행이 가능합니다.)

```
